What's Beyond?

The Beyond Design System (BDS) is built and maintained by Joel Dombek. Beyond includes UI components developed and tested to solve a particular UX problem. BDS is a living document and design test-bed for Joel Dombek Design brand that showcases color palettes, typography and feature-driven design ideas.

Derive-V

The latest version, Derive-V, includes a Light/Dark mode, better handling of media (videos, songs, gifs), animated backgrounds + .svg icons. A full-screen with parallax AI chat prompt is the key new vertical-slice feature. Graphing and data handling is being explored and tested.

Beyond is built on Bootstrap and uses Font Awesome, Google Fonts, Devicons
